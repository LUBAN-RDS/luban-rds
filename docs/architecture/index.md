# 架构设计

本部分详细介绍了 Luban-RDS 的架构设计，包括系统架构、功能架构和设计决策。

## 架构内容

- **[系统架构](./system.md)** - 整体系统架构和模块关系
- **[功能架构](./features.md)** - 各功能模块的详细设计
- **[设计决策](./design.md)** - 重要设计选择的理由和权衡

## 核心架构理念

- **模块化设计**：清晰的模块划分，便于维护和扩展
- **高性能**：基于 Netty 的 NIO 服务器，支持高并发
- **兼容性**：完全兼容 Redis 协议，可直接使用 Redis 客户端
- **可靠性**：支持持久化、备份和恢复机制
- **安全性**：Lua 脚本沙箱，超时控制，操作计数

## 技术栈

| 技术 | 用途 | 版本 |
|------|------|------|
| **Java** | 核心开发语言 | 8+ |
| **Netty** | 网络服务器 | 4.1.x |
| **LuaJ** | Lua 脚本执行 | 3.0.1 |
| **Maven** | 项目管理和构建 | 3.6+ |
| **JUnit** | 单元测试 | 4/5 |

## 模块结构

```
luban-rds/
├── luban-rds-core/          # 核心业务逻辑
├── luban-rds-protocol/      # 协议解析
├── luban-rds-server/        # 网络服务
├── luban-rds-persistence/   # 持久化
├── luban-rds-client/        # Java 客户端
├── luban-rds-common/        # 公共工具
├── luban-rds-bin/           # 启动和基准测试
└── luban-rds-spring-boot-starter/ # Spring Boot 集成
```

## 关键设计特点

1. **内存存储**：高效的内存数据结构，支持多种数据类型
2. **协议兼容**：完全支持 Redis 序列化协议（RESP）
3. **Lua 脚本**：完整的 Lua 脚本支持，与 Redis 兼容
4. **持久化**：支持 RDB 和 AOF 持久化机制
5. **发布订阅**：支持频道订阅和消息广播
6. **Spring Boot 集成**：提供专用的 starter 模块
