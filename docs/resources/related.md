## 相关项目

### 内存数据库
- [Redis](https://redis.io/) - 开源的内存数据结构存储
- [Memcached](https://memcached.org/) - 分布式内存对象缓存系统
- [Hazelcast](https://hazelcast.com/) - 内存数据网格
- [Ignite](https://ignite.apache.org/) - 内存数据网格和分布式数据库

### Redis 客户端
- [Lettuce](https://lettuce.io/) - 高级 Redis 客户端
- [Jedis](https://github.com/redis/jedis) - 流行的 Redis 客户端
- [Redisson](https://redisson.org/) - 分布式和可扩展的 Java 数据结构

### Spring 集成
- [Spring Data Redis](https://spring.io/projects/spring-data-redis) - Spring 框架的 Redis 集成
- [Spring Cache](https://docs.spring.io/spring-framework/docs/current/reference/html/integration.html#cache) - Spring 的缓存抽象

### 监控与管理
- [Redis Commander](https://github.com/joeferner/redis-commander) - Redis 管理界面
- [Redis Insight](https://redis.com/redis-enterprise/redis-insight/) - Redis GUI 工具
- [redis-stat](https://github.com/junegunn/redis-stat) - Redis 统计监控工具
