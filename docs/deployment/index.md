## 部署方式

Luban-RDS 支持多种部署方式，适应不同的应用场景：

- **独立服务**：作为独立的 Redis 兼容服务运行
- **嵌入式**：通过 `EmbeddedRedisServer` 嵌入到应用中
- **Docker 容器**：通过 Docker 容器化部署
- **Kubernetes**：在 Kubernetes 集群中部署
- **云服务**：部署到各大云服务提供商

## 环境要求

| 环境 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **操作系统** | Windows 7+, Linux, macOS | Linux (CentOS 7+, Ubuntu 18.04+) |
| **Java 版本** | Java 8+ | Java 11+ |
| **内存** | 512MB | 2GB+ |
| **CPU** | 1 核 | 2 核+ |
| **网络** | 100Mbps | 1Gbps |

## 关键特性

- **简单部署**：提供多种部署方式，适应不同场景
- **灵活配置**：丰富的配置选项，支持定制化部署
- **易于监控**：完善的监控指标和工具集成
- **高可靠性**：支持持久化、备份和恢复
- **安全可控**：支持密码认证、命令限制等安全特性
